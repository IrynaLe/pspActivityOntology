# PSP Activity Ontology
This repository contains the development drafts for the PSP Activity Ontology.
A human-readable version is available at https://taxonomy.ceitec.vut.cz/pspact/pspActivity.html and is updated on a weekly basis.

The current TTL file defines the hierarchy of PSP activities along with their associated features and measured parameters. Class annotations are currently limited to "rdfs:label" and will be completed in a future update.
Classes adopted from existing materials science ontologies will be assigned corresponding IRIs and comprehensive annotations.
