# PSP Activity Ontology
This repository contains a draft version of the PSP Activity Ontology.
A human-readable version is available at https://taxonomy.ceitec.vut.cz/pspact/pspActivity.html and is updated weekly.

The 20260414_PSPactivity.ttl file contains the hierarchy of PSP activities along with their associated features and measured parameters. Class annotations are currently limited to "rdfs:label" and will be completed in future updates.<br>
Classes adopted from existing materials science ontologies will be assigned corresponding IRIs and comprehensive annotations.
